# NCAA Transfer Portal Analysis

## Overview
This project analyzes how NIL and transfer-rule changes affected player movement, team continuity, and conference dynamics in men's NCAA basketball.

## Research Questions
- Are players transferring more afer 2021?
- Are players stayng fewer years at one school?
- Which conferences gain and lose the most players?
- which teams rely most on transfer-driven roster construction?

## Why This Project
I wrote about NIL in 2021 when the rule first changed. This project revisits those
early expectations and compares them with what actually happened in college basketball.
